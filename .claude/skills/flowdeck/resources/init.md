# init - Save Project Settings (Deprecated)

> **Deprecated:** `flowdeck init` is an alias for `flowdeck config set`. Use `flowdeck config set` instead. See `resources/config.md` for full documentation.

```bash
# Preferred (new command)
flowdeck config set -w App.xcworkspace -s MyApp -S "iPhone 16"

# Still works (deprecated alias)
flowdeck init -w App.xcworkspace -s MyApp -S "iPhone 16"
```

**Options:** Same as `flowdeck config set` - see `resources/config.md`.
